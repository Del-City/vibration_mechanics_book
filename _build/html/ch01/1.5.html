任意激励载荷下作用下的强迫振动

1.5. 任意激励载荷下作用下的强迫振动#

本节讨论单自由度系统在任意一般性荷载（例如图2-23中所示一实测荷载曲线）作 用下的强迫振动问题。

在求解这个问题之前，先来看一种特殊情况。

一个初始静止的单自由度系统，在初始时刻受到一个单位脉冲\(\delta \left ( t \right )\)的作用，求其运动规律。 \(\delta \left ( t \right )\)可看做在一个非常短暂的时间间隔\(\mu\)内作用一个常量力\(\frac{1}{\mu} \)。于是该系统的运动方程为

(1.20)#\[\begin{split} \left\{\begin{matrix} m\ddot{x}\left ( t \right )+ c\dot{x}\left ( t \right )+ kx \left ( t \right )=\delta \left ( t \right ) \\ x \left ( 0 \right )= 0;\dot{x}\left ( 0 \right )= 0 \end{matrix}\right. \end{split}\]

应用动量定理\(m\bigtriangleup \upsilon = p\bigtriangleup t= \frac{1}{\mu} \mu= 1\)，可以将初始时刻的单位脉冲转化为在\(t= \mu \)时刻的速度，即\(\bigtriangleup \upsilon = \frac{1}{m}\)。

因\(\mu\)非常微小，所以系统在\(t= \mu \)时可以视为仍处于初始状态，于是运动方程(1.20)转化为以下形式

(1.21)#\[\begin{split} \left\{\begin{matrix} m\ddot{x}\left ( t \right )+ c\dot{x}\left ( t \right )+ kx \left ( t \right )=0 \\ x \left ( 0 \right )= 0;\dot{x}\left ( 0 \right )= \frac{1}{m} \end{matrix}\right. \end{split}\]

这是一个初始位移为\(0\)，初始速度为\(\frac{1}{m} \)的自由振动问题，它的解记作\(x\left ( t \right )=h\left ( t \right ) \) ，由式eq:1-38可求得

(1.22)#\[\begin{split} \left\{\begin{matrix} h\left ( t \right )= \frac{1}{\omega _{\varepsilon } } e^{- \zeta\omega _{0}t\sin\omega _{\varepsilon }t } \left ( t\ge 0 \right ) \\ 0\left ( t< 0 \right ) \end{matrix}\right. \end{split}\]

式中，\(\omega _{0} = \sqrt{\frac{k}{m} } \)、\(\zeta =\frac{c}{2\sqrt{mk} } \)分别为系统的自振圆频率与阻尼比；\(\omega _{\varepsilon } =\omega _{0} \sqrt{1-\zeta ^{2} } \)为系统有阻尼自振圆频率。

\( h\left ( t \right ) \)是在初始时刻作用单位脉冲而使单自由度系统产生的响应，所以称之为脉冲响应函数（Impulse Response Function）。 在时间间隔\(\left [ 0,t \right ] \)内作用的任意外载\(F_{e} \left ( t \right ) \)，可以看做是在该时间间隔内的大量脉冲\(F_{e} \left (\tau \right ) \mathrm{d}\tau \)的叠加，如图2-24所示。由每一个这种冲量而使系统产生的响应是\(F_{e} \left (\tau \right ) \mathrm{d}\tau h\left ( t- \tau \right ) \)它仅在时刻\(\tau \)之后才起作用。所以力\(F_{e} \left ( t \right ) \)在时刻\(t\)的总响应为

(1.23)#\[ x\left ( t \right ) = \int_{0}^{t} F_{e} \left (\tau \right ) h\left ( t- \tau \right )\mathrm{d}\tau \]

由于在小于\(0\)的时间没有力作用，在大于\(t\)的时间力还没有作用上去，因此以上积分的上下限可以改为\(+ \infty\)和\(- \infty\)，亦即

(1.24)#\[ x\left ( t \right ) = \int_{- \infty }^{+ \infty } F_{e} \left (\tau \right ) h\left ( t- \tau \right )\mathrm{d}\tau \]

作变量代换\(\theta = t- \tau \)可得

(1.25)#\[ x\left ( t \right ) = \int_{- \infty }^{+ \infty } F_{e} \left (t- \theta \right ) h\left (\theta \right )\mathrm{d}\theta \]

式(1.23)~式(1.25)称为杜阿梅尔积分（Duhamel’s Integral）或卷积积分（Convolution Integral），它们都是计算结构瞬态响应十分有用的工具。 事实上，如果在\(t< 0\)时就已经有力作用，式(1.24)或式(1.25)仍然是成立的，而式(1.23)却不能用了。 一般初始条件下，单自由度系统的响应可表示为

(1.26)#\[ x\left ( t \right ) =e^{-\zeta \omega _{0}t } \left ( \frac{\dot{x}_{0}+ \zeta \omega _{0} x_{0} }{\omega _{\varepsilon } }\sin\omega _{\varepsilon }t+x_{0} \cos\omega _{\varepsilon }t \right ) + \int_{- \infty }^{+ \infty } F_{e} \left (\tau \right ) h\left ( t- \tau \right )\mathrm{d}\tau \]

若不考虑阻尼，上式可退化为

(1.27)#\[ x\left ( t \right ) =\frac{\dot{x}_{0}}{\omega _{0} } \sin\omega _{\varepsilon }t+x_{0} \cos\omega _{\varepsilon }t + \int_{- \infty }^{+ \infty } F_{e} \left (\tau \right ) h\left ( t- \tau \right )\mathrm{d}\tau \]

由以上讨论可知，任何外载\(F_{e} \left ( t \right ) \)都可以通过划分为微小的短时间脉冲，借助于\(h \left ( t \right ) \)在时间域的积分（即运用叠加原理）而获得系统的总响应。这种方法通常称为时域法（Time Domain Method）。时域法可用来计算任何线性单自由度系统对任意输入的响应。但有时用频域法（Frequency Domain Method）分析更为方便。频域分析在概念上和1.3节所介绍的周期荷载分析相似，即：首先把荷载展开为简谐作用分量，然后计算系统对每个作用分量的响应，最后再将各简谐分量响应叠加起来得到系统的总响应。数学上已经证明，对于一个时间函数\(F_{e} \left ( t \right ) \)，如果积分\(\int_{- \infty }^{+ \infty }\left | F_{e} \left (t \right ) \right | \mathrm{d}t\)为有限值［即\(F_{e} \left ( t \right ) \)为绝对可积的］，则可将\(F_{e} \left ( t \right ) \)变换为

(1.28)#\[ F_{e} \left ( t \right ) = \frac{1}{2\pi} \int_{-\infty }^{+\infty}P\left ( \omega \right )e^{i\omega t} \mathrm{d}\omega \]

其中

(1.29)#\[ P\left ( \omega\right )=\int_{-\infty }^{+\infty}F_{e} \left ( t \right )e^{- i\omega t} \mathrm{d}t \]

式(1.28)~式(1.29)通常称为逆傅里叶变换（Inverse Fourier Transform）和正傅里叶变换（Direct Fourier Transform）。 与傅里叶级数表达式(1.11)相类似，式(1.28)可理解为将一个任意荷载用无穷个简谐分量的和来表示（傅里叶积分），式中的\(\frac{P\left ( \omega\right )}{2\pi }\)定义了频率为\(\omega\)处单位\(\omega\)的荷载分量的幅值。因此乘以频响函数\(H\left ( \omega\right )\)，就可得出频率为\(\omega\)时单位\(\omega\)的响应分量幅值。因此，在整个频率范围内对这些响应分量求和就可得到总的稳态响应，即

(1.30)#\[ x\left ( t \right )=\frac{1}{2\pi} \int_{-\infty }^{+\infty}H\left ( \omega\right )P\left ( \omega\right )e^{i\omega t} \mathrm{d}\omega \]

为了应用频域法，必须借助式(1.29)计算出给定荷载的简谐分量\(P\left ( \omega\right )\)以及由式(1.15)或(1.16)给出的频响函数。只有当荷载函数能进行傅里叶积分变换时，才能有效地利用频域分析方法。即便如此，计算最后所得的积分也可能是非常麻烦的。在实际应用中，往往是基于离散傅里叶变换（Discrete Fourier Transform，DFT）的概念，采用数值方法进行计算。快速傅里叶变换（Fsst Fourier Transform，FFT）就是一种有效的方法，它的基本概念可在一般的数值方法教科书中查到。 式(1.24)和式(1.30)都可表示单自由度系统的稳态响应，二者是等价的，即

(1.31)#\[ \frac{1}{2\pi} \int_{-\infty }^{+\infty}H\left ( \omega\right )P\left ( \omega\right )e^{i\omega t} \mathrm{d}\omega=\int_{- \infty }^{+ \infty } F_{e} \left (\tau \right ) h\left ( t- \tau \right )\mathrm{d}\tau \]

由式(1.29)，上式可变换为

(1.32)#\[ \frac{1}{2\pi} \int_{-\infty }^{+\infty}H\left ( \omega\right )\left [ \int_{-\infty }^{+\infty}F_{e} \left ( \tau \right )e^{- i\omega \tau } \mathrm{d}\tau \right ] e^{i\omega t} \mathrm{d}\omega=\int_{- \infty }^{+ \infty } F_{e} \left (\tau \right ) h\left ( t- \tau \right )\mathrm{d}\tau \]

进一步化为

(1.33)#\[ \int_{-\infty }^{+\infty}F_{e} \left ( \tau \right )\left [ \frac{1}{2\pi} \int_{-\infty }^{+\infty}H\left ( \omega\right )e^{i\omega \left ( t -\tau \right ) } \mathrm{d}\omega \right ] \mathrm{d}\tau =\int_{- \infty }^{+ \infty } F_{e} \left (\tau \right ) h\left ( t- \tau \right )\mathrm{d}\tau \]

令\(u=t-\tau \)，由上式可看出

(1.34)#\[ h\left ( u\right ) =\frac{1}{2\pi}\int_{-\infty }^{+\infty} H\left ( \omega\right ) e^{i\omega u}\mathrm{d}\omega \]

即

(1.35)#\[ h\left ( t\right ) =\frac{1}{2\pi}\int_{-\infty }^{+\infty} H\left ( \omega\right ) e^{i\omega t}\mathrm{d}\omega \]

在1.4节中，单自由度系统在单位复荷载\(e^{i\omega t}\)作用下的稳态响应为

(1.36)#\[ x\left ( t \right ) =H\left ( \omega\right ) e^{i\omega t} \]

这个单位复荷载激励起的结构稳态响应也可借助杜阿梅尔积分来计算，即

(1.37)#\[ x\left ( t \right ) = \int_{- \infty }^{+ \infty } e^{i\omega \left (t-\theta \right )} h\left (\theta \right )\mathrm{d}\theta \]

比较式(1.36)与式(1.37)可知

(1.38)#\[ H\left ( \omega\right ) e^{i\omega t}= \int_{- \infty }^{+ \infty } e^{i\omega \left (t-\theta \right )} h\left (\theta \right )\mathrm{d}\theta \]

将两边的公共因子\(e^{i\omega t}\)消去，就得到

(1.39)#\[ H\left ( \omega\right ) = \int_{- \infty }^{+ \infty } h\left (\theta \right ) e^{-i\omega \theta } \mathrm{d}\theta \]

即

(1.40)#\[ H\left ( \omega\right ) = \int_{- \infty }^{+ \infty } h\left (t \right ) e^{-i\omega t } \mathrm{d}t \]

由式(1.35)和式(1.40)可知，脉冲响应函数\(h\left (t \right )\)和频率响应函数\(H\left ( \omega\right )\)之间构成傅氏变换对。

Note

【例1-9】一不计阻尼的单自由度系统，初始静止，受简谐扰力\(F_{e} \left ( t \right ) =P\cos \theta t\)作用。试用杜阿梅尔积分求它的响应。 解：对于无阻尼系统，脉冲响应函数由式(1.22)可写为

\[ \begin{aligned} h\left ( t \right )= \left\{\begin{matrix} \dfrac{1}{m\omega}\sin \omega t \left ( t\ge 0 \right ) \cr 0\left ( t< 0 \right ) \end{matrix}\right. \end{aligned} \tag{1} \]

进而由式(1.27)可求得系统的响应为

\[ \begin{aligned} x\left ( t \right ) &= \int_{- \infty }^{+ \infty } F_{e} \left (\tau \right ) h\left ( t- \tau \right )\mathrm{d}\tau= \frac{P}{m\omega } \int_{0}^{t} \cos \theta \tau \sin \omega \left ( t- \tau \right )\mathrm{d}\tau \cr &= \frac{P}{m\left ( \omega ^{2} - \theta ^{2} \right ) }\cos \theta t- \frac{P}{m\left ( \omega ^{2} - \theta ^{2} \right ) }\cos \omega t \end{aligned} \tag{2} \]

应用傅里叶变换进行频域分析时，要求荷载函数\(F_{e} \left ( t \right ) \)在\(\left ( - \infty , + \infty \right ) \)内满足绝对可积等较强的条件。这样的条件，即使对于很简单的函数，如常数、多项式和三角函数等，都不能满足。拉普拉斯（Laplace）变换可以避免这样的困难。和傅里叶变换一样，拉普拉斯变换也是一种常用的研究线性系统的积分变换方法。它的严格数学定义和性质可参考有关的数学书籍，这里只简单介绍应用它来求解线性系统的动力响应。 设函数\(f\left ( t \right ) \)满足条件：

  1. 在\(t\ge 0\)的任一有限区间上分段连续可导；

  2. 当\(t\to+ \infty \)时,\(f\left ( t \right ) \)的增长速度不超过某个指数函数，亦即存在某个常数\(M\)及\(\sigma _{0} \ge 0 \),使得\(\left | f\left ( t \right ) \right | \le Me^{\sigma _{0} t} \left ( 0< t< + \infty \right )\) 其中，\(\sigma _{0} \)称为\(f\left ( t \right ) \)的增长指数。则称

(1.41)#\[ L\left ( p \right ) = \mathcal{L}\left [ f\left ( t \right ) \right ] = \int_{0}^{+ \infty } f\left ( t \right ) e^{- pt}\mathrm{d}t \]

为函数\(f\left ( t \right ) \)的拉氏（拉普拉斯）变换，而称

(1.42)#\[ f\left ( t \right ) = \mathcal{L} ^{- 1} \left [ L\left ( p \right ) \right ]= \frac{1}{2\pi i}\int_{\sigma - i\infty }^{\sigma + i\infty}L\left ( p \right ) e^{pt}\mathrm{d}p \]

为复变函数\( L\left ( p \right )\)的逆拉氏变换。拉普拉斯变换存在的条件要比傅里叶变换弱很多，描述工程问题的大多数荷载函数都能够满足。在实际工作中，简单函数的拉普拉斯变换可查找拉普拉斯变换表。

Note

【例1-10】应用拉普拉斯变换求解单自由度系统在任意荷载\(F_{e} \left ( t \right )\)作用下的响应。设初始条件为\(x\left ( 0 \right ) = x_{0}\)及\(\dot{x} \left ( 0 \right ) = \dot{x}_{0} \)。 解：单自由度系统在任意荷载\(F_{e} \left ( t \right )\)作用下的运动方程为式eq:1-68的形式，即

\[ \ddot{x} \left ( t \right ) + 2\zeta \omega _{0} \dot{x}\left ( t \right )+ \omega _{0}^{2}x\left ( t \right )= \frac{1}{m} F_{e} \left ( t \right ) \tag{1} \]

将式(1)两边做拉普拉斯变换

\[ \mathcal{L} \left [ \ddot{x} \left ( t \right ) + 2\zeta \omega _{0} \dot{x}\left ( t \right )+ \omega _{0}^{2}x\left ( t \right ) \right ]= \frac{1}{m}\mathcal{L} \left [ F_{e} \left ( t \right ) \right ]=\frac{1}{m}F\left ( p \right ) \tag{2} \]

其中

\[ \mathcal{L} \left [ \dot{x}\left ( t \right ) \right ]=\int_{0}^{+\infty }e^{-pt}\dot{x}\left ( t \right )\mathrm{d}t=e^{-pt}x\left ( t \right ) \Bigg|_{0}^{+\infty}+ p\int_{0}^{+\infty}e^{-pt}x\left ( t \right ) \mathrm{d}t = pX\left ( p \right )- x_{0} \tag{3} \]

同理

\[ \mathcal{L} \left [ \ddot{x}\left ( t \right ) \right]= p^{2}X\left ( p \right ) - px_{0} -\dot{x}_{0} \tag{4} \]

因此，式(2)可写为

\[ X\left ( p \right )= \frac{F\left ( p \right )}{m\left ( p^{2}+ 2\zeta \omega _{0}p+ \omega _{0}^{2}\right)}+\frac{p+2\zeta \omega _{0}}{p^{2}+ 2\zeta \omega _{0}p+ \omega _{0}^{2}}x_{0}+\frac{1}{p^{2}+ 2\zeta \omega _{0}p+ \omega _{0}^{2}}\dot{x}_{0} \tag{5} \]

将上式右端化为简单的分式，然后利用卷积定理和拉氏变换表即可求出\(X \left (p \right )\)的拉普拉斯逆变换

\[ \begin{aligned} x(t) = &\mathcal{L}^{-1}\left[ X(p) \right] = e^{-\zeta \omega_{0}t} \left( \frac{\dot{x}_{0} + \zeta \omega_{0} x_{0}}{\omega_{\varepsilon}} \sin\omega_{\varepsilon}t + x_{0} \cos\omega_{\varepsilon}t \right) + \cr &\frac{1}{m\omega_{\varepsilon}} \int_{-\infty}^{+\infty} F_{e}(\tau) e^{-\zeta \omega_{0}(t - \tau)} \sin\omega_{\varepsilon}(t - \tau) \,\mathrm{d}\tau \end{aligned} \tag{6} \]

可以看出，式(6)与式(1.26)是一致的。
